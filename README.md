# Databricks Data Engineer Associate

Five-Friday client enablement program for Databricks Certified Data Engineer Associate preparation.

Learner-facing materials live in the weekly Markdown files, `slides/`, `datasets/`, `notebooks/`, and `cicd/`.

Instructor-only quizzes, mock exams, and notebook solutions live locally under `instructor_private/`. That folder is intentionally ignored by git so learners do not see answers or assessment material in GitHub.
